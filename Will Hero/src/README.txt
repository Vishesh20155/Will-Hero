In order to run the code for the evaluation of second deadline do the following:

1. Run the HomeScreenController file and then there will be 2 options to go to next screen:
    1.1 Click the play button to go to the login screen. Close the Login button to go to the Main Game Screen.
            On the Main game screen, click anywhere to notice a few animations
            Click on the Settings icon to open pause menu
    1.2 Click the Continue Saved Game to see a hypothetical list of saved games

2. Final screen is the Game Over Screen to evaluate that run the GameOverController.java file
    Add VM options for each controller file as well

Just close each screen to go back to the previous screen.
